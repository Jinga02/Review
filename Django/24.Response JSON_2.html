Django shortcuts functions

  • get_object_or_404()
    • 모델 manager objects에서 get()을 호출하지만,
      해당 개체가 없을 땐 기존 DoesNotExist예외 대신 Http404를 raise한다.
  • get_list_or_404()
    • 모델 manager objects에서 filter()의 결과를 반환하고, 해당 객체 목록이 없을 땐 Http404를 raise한다.
  • 서버가 적절한 예외 처리를하고 클라이언트에게 올바른 에러를 전달하는 것은 중요한 요소이다.
