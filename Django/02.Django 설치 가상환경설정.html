django 기본설치

  1. 빈 폴더 만들기
  2. 새 터미널 -> Git bash열기
  3. pip install django==3.2.18 # 장고 설치
  4. django-admin startproject firstpjt프로젝트이름 # firstpjt안에 firstpjt장고프로젝트 생성
  5. cd firstpjt/ # 현재 디렉토리를 변경
  6. django-admin startapp my_app # firstpjt안에 my_app앱 생성
    app 만들고 난 후에 settings.py 파일에 installed_apps 에 만든 app을 추가
    app이란? https://cheershennah.tistory.com/5
  7. python manage.py runserver #를 통한 서버실행
    manage.py(사이트 관리를 도와주는 역할/django프로젝트와 다양한 방법으로 상호작용하는 커맨드라인 유틸리티, 스크립트)
  8. ctrl+c를 누르면 서버종료

가상환경 사용하기

  1. python -m venv venv(가상환경 이름) # 가상환경 생성
  2. source venv/Scripts/activate # 가상환경 구동
  3. pip freeze > requirements.txt # 패키지 종속성 관리(내 환경에 어떤 패키지들을 사용하고 있는지 함께 넘겨주기 위해)
  4. pip install -r requirements.txt # 내가 사용하고 있는 패키지 한번에 install

요청과 응답

  1. setting.py
    INSTALLED_APPS안에 생성한 app추가
  2. 생선한 장고프로젝트 폴더안에 templates폴더생성 => 폴더(이름 원하는걸로)생성 => 파일이름.html 생성 => html문 작성
  3. view.py
    def index(request):
         return render(request, "templates폴더안에 생성한 폴더/폴더 안의 불러올 파일")
  4. urls.py => from 불러올 app이름 import 불러올파일(views)
    urlpatterns => path('주소이름/', 불러올 파일.함수이름)
