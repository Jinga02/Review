UserCreationForm(회원가입)

주어진 username과 password로 권한이 없는 새 user를 생성하는 ModelForm

  • 3개의 필드(username(from the user model), password1, Password2)를 가진다.

  • 회원 가입 페이지 작성하기

    1. accounts의 urls.py에 path 추가.
    2. 회원가입 html 만들기.
    3. 회원가입 함수 만들기.
    4. 위와 같이작성하면 에러 발생하기때문에 forms 수정
    5. 회원가입 후 바로 로그인 처리

회원탈퇴

  • 회원탈퇴 구현하기

    1. accounts의 urls.py에 path 추가.
    2. 회원탈퇴 함수 만들기.
    3. 회원탈퇴 버튼 만들기

회원정보 수정

  • UserChangeForm built-in form을 사용
  • 사용자의 정보 및 권한을 변경하기 위해 admin 인터페이스에서 사용되는 ModelForm

  • 회원정보 수정 구현하기

    1. urls.py에 path추가.
    2. 회원정보 함수 만들기
    3. 회원정보 form 수정

PasswordChangeForm(비밀번호 변경)

  • 비밀번호 변경 구현하기

    1. urls.py path추가
    2. 비밀번호 변경 함수 만들기
    3. 비밀번호 변경 import 수정
    4. 비밀번호 변경 html 만들기

is_authenticated

  • 사용자가 인증 되었는지 여부를 확인 할 수 있는 방법
  • 일반적으로 request.user에서 이 속성을 사용한다.(request.user.is_authenticated)
  • 권한과는 관련이 없으며, 사용자가 활성화 상태이거나 유효한 세션을 가지고 있는지도 확인하지 않는다.
