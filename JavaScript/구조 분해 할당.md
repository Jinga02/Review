    <h2>구조 분해 할당(Destructuring Assignment)</h2>
      <ul>
          <li>배열 또는 객체의 속성을 해체하여 개별 변수로 할당하는 방법이다.</li>
          <li>이를 통해 코드를 간결하게 작성하고 가독성을 높일 수 있다.</li>
          <li>구조 분해 할당은 배열 또는 객체에서 필요한 값만 추출하여 변수에 할당하는 유연한 방법이다.</li>
      </ul>
