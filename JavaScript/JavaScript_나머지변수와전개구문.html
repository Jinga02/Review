JavaScript의 전개 구문과 나머지 변수

전개 구문 (Spread Syntax)

전개 구문은 배열이나 객체 리터럴 내에서 다른 배열이나 객체의 요소를 펼쳐서 복사하거나 결합하는 방법을 제공한다.

나머지 변수 (Rest Parameters)

나머지 변수는 함수 정의에서 가변 인수를 나타내는 방법으로 사용됩니다. 함수에 전달된 모든 인수를 배열로 수집한다.
