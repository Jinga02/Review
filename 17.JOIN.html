JOIN이란?

  • 두 개 이상의 테이블을 서로 연결하여 데이터를 검색할 때 사용하는 방법.
  • 두 개의 테이블을 마치 하나의 테이블인 것처럼 보여준다.

INNER JOIN(내부 조인)

  • 두 테이블을 연결할 때 가장 많이 사용하는 조인이며, 그냥 조인이라고 부르면 내부 조인을 의미한다.

    • SELECT 조회할 컬럼명
    FROM 테이블1, 테이블2
    [WHERE 조건] (<= 선택 )

    SELECT 조회할 컬럼명
    FROM 테이블1
    (INNER) JOIN 테이블2
    ON 테이블1.컬럼 = 테이블2.컬럼
    [WHERE 추가조건] (<= 선택 )

OUTER JOIN

  • 내부 조인은 두 테이블에 모두 데이터가 있어야만 결과가 나오지만,
    외부 조인은 한쪽에만 데이터가 있어도 결과가 나온다.

    • LEFT JOIN : 좌측 테이블 데이터에 추가로 우측 테이블 데이터를 조인하는 문법
    users의 id와 photos의 user_id가 같은 photos filename과 users nickname을 출력하라

    SELECT photos.filename, users.nickname
    FROM photos LEFT JOIN users ON users.id = photos.user_id

    RIGHT JOIN : LEFT JOIN의 반대이며, 우측 테이블 데이터 모두와 좌측 테이블 데이터를 합치는데 조건을 만족하는 데이터만 합치는 것이다.

    FULL JOIN : 좌측, 우측 테이블 상관없이 모든 데이터를 가져오는데 데이터가 빈 값이라면 NULL로 처리해서 가져온다.
